# Employee Management System

This project is a **Java console-based Employee Management System** that helps manage employee records using Java Collections and File Handling. The system allows users to store, update, search, sort, and analyze employee data efficiently.

The goal of this project is to demonstrate the use of important Java concepts such as **ArrayList, HashMap, file handling, exception handling, and object-oriented programming**.

---

## Features

The system provides the following functionalities:

- Add new employees
- View all employee records in a structured table format
- Search employees by **ID, Name, or Department**
- Update employee details
- Delete employee records
- Sort employees by **Salary, Name, or Department**
- Generate **Salary Reports and Department Reports**
- Automatic saving of data using file persistence
- Input validation for name, salary, and menu choices

---

## Technologies Used

- Java
- Java Collections Framework
- File Handling (Serialization)
- Object-Oriented Programming

## How to Run the Project

1. Make sure **Java JDK** is installed on your system.

2. Open the project folder in **VS Code or any Java IDE**.

3. Compile the project using:


javac -d . src/com/*.java


4. Run the application:


java com.EmployeeManagementSystem


---

## Example Workflow

1. Start the program.
2. Choose an option from the main menu.
3. Perform operations such as adding, searching, or updating employees.
4. The system automatically saves changes to the data file.
5. Reports and sorted results can be viewed anytime.
